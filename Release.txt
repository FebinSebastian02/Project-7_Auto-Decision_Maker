An Automatic decision making program where the computer will chose an Anime for the user based on the mood of the user is created with:-
	- a list to collect few anime names
	- a class and an object
	- functions defined and called
	- a for loop to iterate through the items in list
	- a while loop to run the program continously until the user needs to quit
	- if else conditions to recommend anime suggestions based on the mood
